# 实体字符转字符串

## 在线演示

<code src="./HtmlToString.tsx">实体字符转字符串</code>


## API

```typescript
// html实体字符转字符串
htmlToString(data:string);
```

### 参数

| 参数     | 说明                                                                              | 类型 |
| -------- | --------------------------------------------------------------------------------- | ---- |
| data     | 需要被转义的包含实体字符的字符串                                                                | string  |

### 返回

(string):返回转义完成的字符串
