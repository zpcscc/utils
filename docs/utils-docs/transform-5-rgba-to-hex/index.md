# rgba 转 16 进制

## 在线演示

<code src="./RgbaToHex.tsx">rgba 转 16 进制</code>


## API

```typescript
// rgba颜色 转为 十六进制颜色
rgbaToHex(data:string);
```

### 参数

| 参数     | 说明                                                                              | 类型 |
| -------- | --------------------------------------------------------------------------------- | ---- |
| data     | rgba格式的字符串                                                                | string  |

### 返回

(string):返回十六进制的字符串
