# 16 进制 转 rgba

## 在线演示

<code src="./HexToRgba.tsx">16 进制 转 rgba</code>

## API

```typescript
// 十六进制颜色 转为 rgba颜色 
hexToRgba(data:string);
```

### 参数

| 参数     | 说明                                                                              | 类型 |
| -------- | --------------------------------------------------------------------------------- | ---- |
| data     | 十六进制颜色格式的字符串                                                                | string  |

### 返回

(string):返回rgba颜色格式的字符串
