# 数据校验

## 参数

| 参数名 | 类型                      | 描述                  |
| ------ | ------------------------- | --------------------- |
| data   | <code>any</code>          | <p>需要校验的数据</p> |
| type   | <code>ValidateType</code> | <p>需要校验的类型</p> |

## 返回

<code>boolean</code><p>返回校验的结果, 默认返回 false</p>
