# rgb 转 hsl

## 在线演示

<code src="./transform/rgbToHsl">rgb 转 hsl</code>

## API

```typescript
// rgb颜色 转为 hsl颜色
rgbToHsl(data:string);
```

### 参数

| 参数 | 说明             | 类型   |
| ---- | ---------------- | ------ |
| data | rgb 格式的字符串 | string |

### 返回

(string):返回 hsl 的字符串
