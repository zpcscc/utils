# hex 转 hsl

## 在线演示

<code src="./transform/hexToHsl">hex 转 hsl</code>

## API

```typescript
// hex颜色 转为 hsl颜色
hexToHsl(data:string);
```

### 参数

| 参数 | 说明                 | 类型   |
| ---- | -------------------- | ------ |
| data | hex 颜色格式的字符串 | string |

### 返回

(string):返回 hsl 颜色格式的字符串
