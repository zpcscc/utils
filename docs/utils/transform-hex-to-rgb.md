# hex 转 rgb

## 在线演示

<code src="./transform/hexToRgb">hex 转 rgb</code>

## API

```typescript
// hex颜色 转为 rgb颜色
hexToRgb(data:string);
```

### 参数

| 参数 | 说明                 | 类型   |
| ---- | -------------------- | ------ |
| data | hex 颜色格式的字符串 | string |

### 返回

(string):返回 rgb 颜色格式的字符串
