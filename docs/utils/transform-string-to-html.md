# 字符串转实体字符

## 在线演示

<code src="./transform/stringToHtml">字符串转实体字符</code>

## API

```typescript
// 字符串转html实体字符
stringToHtml(data:string);
```

### 参数

| 参数 | 说明                       | 类型   |
| ---- | -------------------------- | ------ |
| data | 需要转义成实体字符的字符串 | string |

### 返回

(string):返回转义完成的字符串
