# hsl 转 rgb

## 在线演示

<code src="./transform/hslToRgb">hsl 转 rgb</code>

## API

```typescript
// hsl颜色 转为 rgb颜色
hslToRgb(data:string);
```

### 参数

| 参数 | 说明                 | 类型   |
| ---- | -------------------- | ------ |
| data | hsl 颜色格式的字符串 | string |

### 返回

(string):返回 rgb 颜色格式的字符串
