# hex、rgb 转换为 hsl

## 在线演示

<code src="./toColorHsl"></code>

## API

```typescript
// 转为 hsl颜色
toColorHsl(data:string);
```

### 参数

| 参数 | 说明                       | 类型   |
| ---- | -------------------------- | ------ |
| data | hex 、rgb 颜色格式的字符串 | string |

### 返回

(string):返回 hsl 颜色格式的字符串
