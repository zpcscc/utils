## 使用

https://zpcscc.top/utils

## 开发

```shell
# 安装依赖
pnpm install

# 本地开发
pnpm start

# 打包资源库代码
pnpm run build

# 预览打包后的代码
pnpm run build:watch

# 打包文档
pnpm run docs:build
```
